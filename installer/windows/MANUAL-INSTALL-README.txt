Lofi Tape Saturator — Manual Install (Windows x64)

Requirements
- Windows 10 or later (64-bit)
- VST3-compatible DAW (FL Studio, Ableton, etc.)
- Microsoft Edge WebView2 Runtime (installed)

Contents
- Lofi Tape Saturator.vst3 (folder/bundle)
- INSTALLAZIONE-WINDOWS.md (full guide)

Install Steps
1) Close your DAW completely
2) Copy the folder "Lofi Tape Saturator.vst3" to one of these paths:
   - C:\Program Files\Common Files\VST3\
   - %AppData%\Image-Line\FL Studio\Plugins\VST3\ (FL Studio)
   - %UserProfile%\Documents\VST3\ (alternative)
3) Start your DAW and run a plugin rescan, if needed
4) In FL Studio, check Plugin Manager → VST3 → Lofi Tape Saturator

Troubleshooting
- If the plugin appears but the UI is blank:
  - Ensure WebView2 Runtime is installed
  - Verify the bundle contains: Contents\Resources\web\index.html
- If the plugin does not appear:
  - Confirm the folder path is correct
  - Reset blacklist and rescan in FL Studio

Uninstall
- Delete the folder from the install path(s) above
